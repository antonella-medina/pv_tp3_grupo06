## Descripción del Proyecto
Este proyecto es una migración de una plataforma de gestión educativa desarrollada originalmente en HTML/CSS (TP N° 1) a una aplicación funcional en React. 
Se aplicó una arquitectura de componentes, manejo de estado global con useState y servicios lógicos independientes.
Aplicación web desarrollada con React y Vite para la gestión de proyectos educativos.  El proyecto permite visualizar, buscar, agregar y eliminar proyectos utilizando componentes funcionales, hooks y lógica modularizada.
